# Tech-Challenge - FIAP

## Pré-requisitos

- Java 17
- Maven
- Docker
- Docker Compose

## EXECUTAR

1. Construa, inicie e execute os containers Docker:
    ```sh
    docker-compose up --build 
    ```
   OU
   ```sh
    docker-compose up -d 
    ```
